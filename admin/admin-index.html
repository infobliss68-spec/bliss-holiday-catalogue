How to edit prices, descriptions, or add photos

  1. Go to your GitHub repository (bliss-holiday-catalogue)
  2. Navigate to the folder data/excursions/
  3. Click on the JSON file you want to edit (e.g., south1.json for South Tour)
  4. Click the pencil icon (Edit) at the top right
  5. Modify the fields you want (prices, description, etc.)
  6. For photos/videos: upload to assets/excursions/ folder, then reference the path in the media field like: "/assets/excursions/photo1.jpg"
  7. Click Commit changes → Netlify rebuilds automatically in 1 minute ✨
